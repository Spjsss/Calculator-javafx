# JavaFX Calculator

A desktop calculator application built using **JavaFX**, featuring a clean and intuitive graphical interface.  
The application supports basic arithmetic operations and demonstrates event-driven programming and user input handling.



## 🎮 Features
- User-friendly graphical interface  
- Basic arithmetic operations (addition, subtraction, multiplication, division)  
- Button-based input with event handling  
- Input validation for accurate calculations  
- Responsive and clean UI design  



## 🛠️ Technologies Used
- Java  
- JavaFX  



## 🚀 How to Run

### 1. Clone the repository
```bash
git clone [https://github.com/Spjsss/Calculator-javafx.git]
```
### 2. Open the project

Open the project in an IDE such as IntelliJ IDEA, NetBeans, or VS Code

### 3. Configure JavaFX

Make sure JavaFX is installed and added to your project.

Example VM options:
```bash
--module-path /path/to/javafx/lib --add-modules javafx.controls,javafx.fxml
```
### 4. Run the application

Run the main class to start the calculator
